# Backend-project Pau's Playlists 🎶🎧

## Endpoints:

- /playlists  
  _Retrieve all playlists_ 🎵

- /playlists/:id  
  _Retrieve a specific playlist_ 🎼

- /songs  
  _Retrieve all songs in playlists_ 🎹

- /songs/:id  
  _Retrieve a specific song_ 🥁

- /:id/songs  
  _Retrieve all songs from specific playlist_ 🎸
